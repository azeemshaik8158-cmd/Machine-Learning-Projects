# Amazon Reviews Sentiment Analysis

This project is about classifying Amazon product reviews into Positive, Negative or Neutral sentiment using Machine Learning and NLP techniques. The goal is to understand customer opinions better and help businesses improve their products and services using review insights.

---

E-commerce platforms receive millions of reviews every day.  
Manually analyzing them is impossible.

Being able to automatically classify reviews helps in:
- Understanding customer satisfaction
- Identifying negative feedback early
- Improving product quality and user experience
- Supporting data-driven business decisions
